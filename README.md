# Project Succession - Releases

This repository hosts release artifacts for Project Succession.

**Source code is in a private repository.** This repository only contains:
- Installer binaries (via GitHub Releases)
- Version manifest (`latest.json`)

## Download Latest Version

Visit the [latest release](https://github.com/JeremiasMeister/succession-releases/releases/latest) to download installers for:
- Windows (.msi)
- Linux (.AppImage, .deb)

## Version Manifest

The `latest.json` file contains version information used by the app's auto-update checker.
